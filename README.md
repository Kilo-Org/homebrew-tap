# Kilocode Homebrew Tap

This is the official Homebrew tap for the [Kilo CLI](https://kilo.ai/cli), the most complete CLI for agentic engineering.

## Installation

```bash
# Add the tap
brew tap kilocode/homebrew-tap

# Install Kilo
brew install kilocode
```

Or install directly:

```bash
brew install kilocode/homebrew-tap/kilocode
```

## Usage

After installation, you can use the `kilo` command:

```bash
kilo --version
```

## Updating

The formula is automatically updated when new versions of the CLI are released. To get the latest version:

```bash
brew update
brew upgrade kilocode
```

## Development

This tap contains formulas for:

- `kilocode` - The Kilo CLI tool

The formulas are automatically updated via GitHub Actions when new versions are released.
